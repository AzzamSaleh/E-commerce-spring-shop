# Spring Shop

Full-stack e-commerce project built with:

- Angular (Frontend)
- Spring Boot (Backend)
- MySQL (Database)

## Project Structure

- `frontend/` → Angular application
- `backend/` → Spring Boot REST API

## Features

- Product listing
- Product details
- Cart
- Order management
- REST API integration

## Technologies Used

### Frontend
- Angular
- TypeScript
- HTML
- CSS

### Backend
- Spring Boot
- Java
- Spring Data JPA
- Hibernate
- MySQL
